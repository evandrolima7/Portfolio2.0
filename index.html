Good Food - Restaurante

Um site de restaurante desenvolvido para fins de estudo, utilizando HTML, CSS, JavaScript e Bootstrap. O projeto foi criado com foco em responsividade e experiência do usuário, apresentando as seções de cardápio, agendamento de reservas e informações sobre o restaurante. Inclui animações suaves, formulário de contato e integração com o WhatsApp para pedidos.

Portfolio 2.0

Aplicação de meu portfolio pessoal, nele eu destaco os meus projetos e minha habilidade em desenvolvimento web. Meu portfolio reflete as informações mais relevantes sobre mim, pude aplicar meu conhecimento em designer, animacoes e estilos, espero que você goste do meu portfolio.

Titulo Projeto 3

Aplicação web feita com JavaScript que consome a API do Rick and Morty, exibindo informações dos personagens da série como nomes, imagens e status. O projeto utiliza técnicas de requisição assíncrona (fetch API) para carregar os dados em tempo real, proporcionando uma experiência interativa para os fãs da série.

To-do-list React

Um aplicativo de lista de tarefas simples e intuitivo, desenvolvido com HTML, CSS e JavaScript. O usuário pode adicionar, editar e remover tarefas, permitindo uma gestão eficiente das atividades diárias. O projeto inclui funcionalidades como a persistência das tarefas no armazenamento local do navegador, garantindo que as informações sejam mantidas mesmo após a recarga da página. A interface é responsiva e otimizada para diferentes dispositivos, proporcionando uma experiência de usuário agradável. Tecnologias: HTML, CSS, JavaScript, LocalStorage

Controle De Estoque

Aplicação web desenvolvida em JavaScript para gerenciar produtos e preços, voltada para pequenas empresas. O sistema permite cadastrar, editar e excluir produtos, com formatação de valores em Real brasileiro. Oferece uma interface intuitiva para facilitar o controle de estoque de forma simples e eficiente.

Apredendo Inglês

Jogo educativo desenvolvido com HTML, CSS, JavaScript, TypeScript e Angular, focado no aprendizado de inglês. O usuário valida frases corretamente para avançar no jogo, tornando o processo de aprendizado mais interativo e divertido. Ideal para praticar o idioma de forma leve e envolvente.

Creator button

Ferramenta web simples para gerar botões personalizados com diferentes estilos de design. O usuário pode ajustar propriedades como cor, tamanho, borda, e sombra, visualizando as mudanças em tempo real e copiando o código CSS gerado. Ideal para desenvolvedores que desejam criar botões personalizados de forma rápida e prática. Tecnologias: HTML, CSS, JavaScript

Calcule IMC

Aplicação web que permite calcular o Índice de Massa Corporal (IMC) de forma rápida e simples. O usuário insere seu peso e altura, e a ferramenta exibe o valor do IMC junto com uma classificação baseada nas faixas de peso (abaixo do peso, peso normal, sobrepeso, etc.). A interface é responsiva, garantindo uma boa experiência em dispositivos móveis e desktops. Tecnologias: HTML, CSS, JavaScript

Curiodidades Android

Uma landing page informativa que apresenta curiosidades e fatos interessantes sobre o sistema operacional Android. O design é moderno e responsivo, com seções que detalham a história, evolução, e versões do Android. Ideal para entusiastas de tecnologia que querem aprender mais sobre o sistema que domina o mercado mobile. Tecnologias: HTML, CSS, JavaScript
