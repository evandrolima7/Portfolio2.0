Olá! Sou um desenvolvedor front-end em transição de carreira, atualmente cursando graduação em Análise e Desenvolvimento de Sistemas, com previsão de formatura em 2025, estudo programação há mais de um ano. Estou empolgado para compartilhar minha jornada e experiência com você. Comecei a criar uma base de páginas web com HTML e CSS , e rapidamente me aprofundei em estilos avançados usando Sass , Bootstrap e Tailwind CSS. Adoro transformar ideias em designs visualmente atraentes e amigáveis ​​para o usuário.

Minha paixão por criar experiências interativas me levou a aprender JavaScript e TypeScript . Agora posso adicionar funcionalidades incríveis às páginas da web. Trabalhei com Angular e React para construir aplicativos web mais complexos e envolventes. O que me motiva é a busca pela excelência em cada projeto. Além disso, estou sempre ansioso para aprender e colaborar com equipes talentosas

Se você procura um desenvolvedor front-end dedicado, apaixonado por criar interfaces incríveis e prontas para fazer parte do seu tempo, estou à disposição para contribuir com meu conhecimento e habilidades

Good Food - Restaurante

Um site de restaurante desenvolvido para fins de estudo, utilizando HTML, CSS, JavaScript e Bootstrap. O projeto foi criado com foco em responsividade e experiência do usuário, apresentando as seções de cardápio, agendamento de reservas e informações sobre o restaurante. Inclui animações suaves, formulário de contato e integração com o WhatsApp para pedidos.

Controlador De Vendas

Aplicação web para gerenciamento financeiro, desenvolvida com React, Bootstrap e Tailwind. Permite controle de vendas, despesas e estoque, com funcionalidades para adicionar, editar e excluir registros. O sistema também realiza cálculos automáticos de lucro, proporcionando uma visão clara da saúde financeira do negócio.

Titulo Projeto 3

Aplicação web feita com JavaScript que consome a API do Rick and Morty, exibindo informações dos personagens da série como nomes, imagens e status. O projeto utiliza técnicas de requisição assíncrona (fetch API) para carregar os dados em tempo real, proporcionando uma experiência interativa para os fãs da série.

Controle De Estoque

Aplicação web desenvolvida em JavaScript para gerenciar produtos e preços, voltada para pequenas empresas. O sistema permite cadastrar, editar e excluir produtos, com formatação de valores em Real brasileiro. Oferece uma interface intuitiva para facilitar o controle de estoque de forma simples e eficiente.

Apredendo Inglês

Jogo educativo desenvolvido com HTML, CSS, JavaScript, TypeScript e Angular, focado no aprendizado de inglês. O usuário valida frases corretamente para avançar no jogo, tornando o processo de aprendizado mais interativo e divertido. Ideal para praticar o idioma de forma leve e envolvente.
